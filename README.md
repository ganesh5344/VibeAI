# VibeAI 🎧

AI Mood Based Music Recommendation System
https://colab.research.google.com/drive/1q2gPWRQkikIG5aVqlLsXdcRYTlZfIhkn?usp=sharing
## Description

VibeAI is an AI application that recommends music playlists based on user mood and preferred genres.

The system uses natural language processing techniques to detect mood from text input and generates playlist links for YouTube and Spotify.

## Features

- mood detection from text
- multi genre selection
- playlist recommendation
- chatbot interaction
- modern mobile UI design
